# C Career Frontend

Frontend application for the C Career platform built with modern web technologies.

## Description

This project is the frontend client for the C Career application. It provides the user interface for interacting with the backend API, including authentication, job management, and other platform features.

## Prerequisites

Before running the project, make sure you have the following installed:

- Node.js 18+
- npm or yarn
- Git
- VS Code (optional)

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/Neal-Ramos/C_Career-Frontend.git
```

### 2. Go to the project folder

```bash
cd C_Career-Frontend
```

### 3. Install dependencies

```bash
npm install
```

### 4. Configure environment variables

Create a `.env` file and Use This Template.

Example:

```env
VITE_API_URL=http://localhost:5221/
VITE_GOOGLE_CLIENTID=YOUR GOOGLE OAUTH CLIEND ID
```

### 5. Run the development server

```bash
npm run dev
```

The frontend application should now be running locally.
